## Compare mode

This mode will be similar to the first mode, but you will compare metrics between 2 different runs

For each run you will give information on the following:
- distance
- time
- age
- gender
- resting heart rate (if available)
- max heart rate (if available)
- avg heart rate of race (if available)

The for each race we can find the absolute values of these and compare with eachother:
- avg pace (for each)
- vdot
- age grading percentage
  - If we put your run first, we can put their run in terms of your time
  - 
- lactate threshold (for both pace and heart rate)
